# ntua-appathon-project
My repo for the Web Development course in ECE NTUA

Το project θα χρησιμοποιεί τα δεδομένα καιρού από το Web Service 11 και τη βάση MySQL που δίνεται για να πληροφορεί τους χρήστες για τους τερματικούς σταθμούς που μπορούν να φτάσουν από τις λεωφορειακές γραμμές στων οποίων την αφετηρία δεν βρέχει τη στιγμή της αναζήτησης. Για backend θα χρησιμοποιηθεί NodeJS και το Express Framework και για frontend η ReactJS.
